Introducción

GarageLog Desktop es una aplicación de escritorio para la gestión integral de un taller o garaje. Permite administrar vehículos, modelos de vehículos, piezas y mantenimientos de forma sencilla y eficiente.

La aplicación se conecta a un servidor backend que almacena los datos en una base de datos MySQL, garantizando la persistencia y seguridad de la información.

Requisitos del sistema

  • Sistema operativo: Windows 10 o superior
  • Java Runtime Environment (JRE) 17 o superior
  • Servidor backend Spring Boot en ejecución (accesible en http://localhost:8080)
  • Base de datos MySQL

Estructura de la aplicación

La aplicación está organizada en los siguientes módulos principales:

  • Vehículos: pantalla principal con el listado completo de vehículos registrados
  • Modelos: gestión de marcas, modelos y especificaciones técnicas
  • Piezas: catálogo de piezas y recambios disponibles
  • Mantenimientos: registro de operaciones de mantenimiento para cada vehículo

Módulo Vehículos

La pantalla principal de la aplicación muestra el listado completo de vehículos registrados en el sistema. Desde aquí se pueden realizar todas las operaciones CRUD.

La tabla de vehículos muestra los siguientes datos:

  • ID: identificador único del vehículo
  • Matrícula: número de matrícula del vehículo
  • Modelo: marca y modelo del vehículo
  • KM Iniciales: kilometraje inicial registrado
  • Fecha Compra: fecha de adquisición del vehículo

Para actualizar el listado manualmente, pulsa el botón Refrescar.

Crear un vehículo

  1. Pulsa el botón Nuevo Vehículo en la pantalla principal
  2. Selecciona un Modelo del desplegable (debe existir previamente en el catálogo de modelos)
  3. Introduce la Matrícula del vehículo
  4. Introduce los KM Iniciales
  5. Selecciona la Fecha de Compra
  6. Añade cualquier Nota adicional (opcional)
  7. Pulsa Guardar para registrar el vehículo
Asegúrate de que el modelo ya exista en el catálogo. Si no está, créalo desde Gestión Modelos antes de registrar el vehículo.

Editar un vehículo

  1. Selecciona el vehículo en la tabla principal
  2. Pulsa el botón Editar Vehículo
  3. Modifica los campos necesarios en el formulario
  4. Pulsa Guardar para confirmar los cambios

Eliminar un vehículo

  1. Selecciona el vehículo en la tabla principal
  2. Pulsa el botón Eliminar Vehículo
  3. Confirma la operación en el cuadro de diálogo
Al eliminar un vehículo se eliminarán también todos sus mantenimientos asociados.

Módulo Modelos de Vehículo

Este módulo permite gestionar el catálogo de modelos de vehículos. Los modelos se utilizan al crear o editar un vehículo.

Acceso: Desde la pantalla principal, pulsa Gestión Modelos en la barra inferior.

Campos del formulario

  • Marca: fabricante del vehículo (obligatorio)
  • Modelo: nombre del modelo específico (obligatorio)
  • Año: año de fabricación o lanzamiento
  • Combustible: tipo de combustible (Gasolina, Diésel, Híbrido, Eléctrico, GLP)

Crear un modelo

  1. Pulsa Nuevo en la ventana de Gestión de Modelos
  2. Rellena los campos del formulario
  3. Pulsa Guardar

Editar un modelo

  1. Selecciona un modelo en la tabla
  2. Pulsa Editar
  3. Modifica los campos y pulsa Guardar

Eliminar un modelo

  1. Selecciona un modelo en la tabla
  2. Pulsa Eliminar
  3. Confirma la eliminación
No se puede eliminar un modelo que tenga vehículos asociados.

Módulo Piezas

Este módulo gestiona el catálogo de piezas y recambios disponibles en el taller. Las piezas se utilizan al registrar mantenimientos.

Acceso: Desde la pantalla principal, pulsa Gestión Piezas en la barra inferior.

Campos del formulario

  • Nombre: nombre descriptivo de la pieza (obligatorio)
  • Descripción: información adicional sobre la pieza
  • Precio: precio unitario de la pieza (obligatorio)

Crear una pieza

  1. Pulsa Nuevo en la ventana de Gestión de Piezas
  2. Rellena los campos del formulario
  3. Pulsa Guardar

Editar una pieza

  1. Selecciona una pieza en la tabla
  2. Pulsa Editar
  3. Modifica los campos y pulsa Guardar

Eliminar una pieza

  1. Selecciona una pieza en la tabla
  2. Pulsa Eliminar
  3. Confirma la eliminación

Módulo Mantenimientos

Este módulo permite registrar y consultar las operaciones de mantenimiento realizadas a cada vehículo, incluyendo las piezas utilizadas.

Acceso:

  1. Selecciona un vehículo en la tabla principal
  2. Pulsa el botón Ver Mantenimientos

Ventana de mantenimientos

La ventana se divide en dos paneles:

  • Panel superior: lista de mantenimientos del vehículo seleccionado
  • Panel inferior: piezas utilizadas en el mantenimiento seleccionado

Al seleccionar un mantenimiento en la tabla superior, las piezas asociadas se muestran automáticamente en la tabla inferior.

Crear un mantenimiento

  1. Pulsa Nuevo Mantenimiento
  2. Selecciona la Fecha del mantenimiento
  3. Introduce los KM actuales del vehículo
  4. Indica el Coste Total (opcional)
  5. Añade Observaciones si es necesario
  6. En la sección Piezas, selecciona las piezas utilizadas:
    • Elige una pieza del desplegable
    • Indica la cantidad y el precio unitario
    • Pulsa Agregar para añadirla a la lista
  7. Pulsa Guardar para registrar el mantenimiento
El precio unitario se rellena automáticamente al seleccionar una pieza, pero puedes modificarlo si el precio real difiere del catálogo.

Eliminar un mantenimiento

  1. Selecciona el mantenimiento en la tabla superior
  2. Pulsa Eliminar Mantenimiento
  3. Confirma la eliminación

Consejos de uso

Antes de crear un vehículo

Asegúrate de que el modelo del vehículo ya existe en el catálogo de modelos. De lo contrario, créalo primero desde Gestión Modelos.

Precio de piezas en mantenimientos

Al seleccionar una pieza en el formulario de mantenimiento, el precio se autocompleta con el valor del catálogo. Si el proveedor aplicó un precio distinto, puedes ajustarlo manualmente.

Mantenimientos periódicos

Revisa regularmente el kilometraje de los vehículos para planificar los mantenimientos según las recomendaciones del fabricante.

Datos consistentes

Mantén actualizado el catálogo de modelos y piezas. Un catálogo bien organizado facilita el registro de vehículos y mantenimientos.

Refrescar datos

Si trabajas con varios usuarios simultáneamente, usa el botón Refrescar para asegurarte de ver los datos más recientes.

Errores comunes y soluciones

Error de conexión con el servidor

Síntoma: Mensaje de error al iniciar o al realizar operaciones indicando que no se puede conectar con el servidor.

Solución:

  1. Verifica que el servidor backend esté en ejecución
  2. Asegúrate de que el backend se esté ejecutando en http://localhost:8080
  3. Comprueba que no haya un cortafuegos bloqueando la conexión
  4. Reinicia el servidor backend si es necesario

Error al guardar un modelo

Síntoma: Al crear o editar un modelo, aparece un error del servidor.

Solución:

  • Verifica que los campos obligatorios (marca y modelo) no estén vacíos
  • Comprueba que el año sea un número válido
  • Asegúrate de que el servidor backend esté funcionando correctamente

Error al guardar un mantenimiento

Síntoma: El mantenimiento no se guarda y aparece un error.

Solución:

  • Verifica que la fecha esté seleccionada
  • Comprueba que los KM sean un número válido
  • Asegúrate de que el vehículo asociado exista en la base de datos

La tabla no muestra datos

Síntoma: Una tabla aparece vacía cuando debería mostrar registros.

Solución:

  • Pulsa el botón Refrescar para recargar los datos
  • Verifica que el servidor backend esté ejecutándose
  • Comprueba que la base de datos contenga registros

Error al eliminar un modelo

Síntoma: No se puede eliminar un modelo.

Solución: Solo se pueden eliminar modelos que no tengan vehículos asociados. Elimina primero los vehículos que usan ese modelo o asígnales otro modelo diferente.

La aplicación no inicia

Síntoma: Al ejecutar la aplicación, esta no se abre o muestra un error de Java.

Solución:

  • Verifica que Java 17 o superior esté instalado: java -version
  • Asegúrate de que el archivo JAR no esté dañado
  • Ejecuta la aplicación desde la terminal para ver los mensajes de error detallados
